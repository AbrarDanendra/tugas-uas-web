Tugas UAS Web - Abrar



Repository ini dibuat buat menampung source code website statis dalam rangka memenuhi nilai tugas akhir (UAS).



Stack \& Tools Yang Dipake

1. Frontend: HTML5 \& CSS3 (Biar tampilannya responsif dan enteng)

2\. Version Control: GitHub (Buat backup dan kolaborasi kodingan)

3\. Hosting Platform: Cloudflare Pages (Arsitektur Serverless biar gak ribet urusan spek VPS)

4\. Domain Custom: \[devbrar.my.id](https://devbrar.my.id)



Sistem Kerja (Workflow)

Website ini udah pakai sistem CI/CD otomatis. Jadi kalau ada revisi atau update kodingan di folder laptop, kita tinggal push aja ke GitHub ini, nanti websitenya otomatis ke-update sendiri di domain domain kita tanpa perlu restart server manual. Plus, udah dapet setup SSL/HTTPS gratis dari Cloudflare jadi webnya aman dan gak ada status 'Not Secure'.
